# Cross-Post Adapt

Transform a single content idea into platform-native posts that do NOT look like the same post copied four times. X threads do not become LinkedIn posts. LinkedIn posts do not become Instagram captions. Each version earns its place on its platform.

## When to Use

- Founder wrote one post and says "put this everywhere"
- Content calendar calls for the same topic across multiple platforms
- Repurposing an existing post to extend its reach
- Building a coordinated multi-platform campaign around one message
- Weekly content batch: write once, adapt intelligently

## Quick Reference

| Source Format | X Adaptation | LinkedIn Adaptation | IG Adaptation | TikTok Adaptation |
|--------------|-------------|--------------------|--------------|-----------------|
| X thread (6-10 tweets) | (source) | Condense to 900-1200 char story post or carousel | Extract 1-2 insights as quote cards + caption | Extract 1 insight as video script (30-60s) |
| LinkedIn post (900-1200 chars) | Condense to thread (4-6 tweets) | (source) | Pull key insight + visual for carousel or story | Pull 1 insight for video (~30s) |
| IG caption (125-150 words) | Extract strongest line as a short post or 3-4 tweet thread | Expand into 900-char story post | (source) | Turn into video script |
| TikTok script | Condense to short post or 3-4 tweet thread | Expand into professional-framed post | Adapt to Reels format | (source) |

## Procedure

1. Identify the source content and its platform of origin.

2. Extract the core message (one sentence — the thing every adaptation must preserve).

3. For each target platform, determine:
   - What format works best for this message on this platform? (Thread, short post, carousel, video)
   - What needs to change structurally? (X is punchy, LinkedIn is structured, IG is visual-led, TikTok is video-first)
   - What needs to change tonally? (More/less casual, more/less professional, more/less emoji)

4. Write each platform adaptation from scratch:
   - Do NOT copy-paste and tweak. Write fresh.
   - Preserve the core message but express it in the platform's native language
   - New hooks for each platform (what works on X doesn't work on LinkedIn)
   - Platform-specific CTAs

5. Apply structural transformations:
   - X thread → LinkedIn: collapse multiple tweets into flowing narrative. Add depth.
   - LinkedIn post → X thread: break long paragraphs into tweet-sized chunks. Sharpen hooks.
   - Any text → IG: determine what visual carries the message. Caption supports, doesn't lead.
   - Any text → TikTok: determine 1-3 visual/audio hooks. Script around them.

6. Check for "copy-paste signals":
   - Same hook across platforms (dead giveaway)
   - Same emoji patterns (especially if emoji usage differs per platform)
   - Same hashtag strategy (LinkedIn hashtag density should not match IG)
   - Same paragraph structure (LinkedIn paragraphs != X line breaks)

7. Sequence cross-platform posting:
   - Post to primary platform first
   - Wait 2-4 hours before posting adapted version to secondary platform
   - Adjust if the primary version is performing exceptionally well (adapt the adaptation based on real-time feedback)

## Pitfalls

- "We'll just change the line breaks and call it adapted" — audiences on multiple platforms will see both and notice
- Same CTA across all platforms (LinkedIn audience wants "agree?" — TikTok audience wants "follow for part 2")
- Adapting a format that fundamentally doesn't work on the target platform (LinkedIn article → TikTok video is a creative stretch, not an adaptation)
- Posting all adapted versions at the same time (cross-platform followers see clone content in their feed)
- Forgetting that some content is genuinely single-platform (not everything belongs everywhere)

## Verification

Put the adaptations side by side. If you blocked out the platform label, could you tell which platform each was written for? If not, the adaptations are too similar. Each should feel native to its platform.
