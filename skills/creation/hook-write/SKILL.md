# Hook Write

Generate high-performing hooks for any content type and platform. The hook is the single most important element of any post — if nobody stops scrolling, the rest doesn't matter. This skill produces multiple variations so the best one can be tested and selected.

## When to Use

- Before writing a post or thread (hook-first writing)
- A/B testing hook variations (deploy 2 versions, measure which wins)
- Existing post underperformed — diagnose if the hook was the problem
- Content calendar needs hooks drafted in advance
- Founder says "I have an idea" but doesn't know how to open

## Quick Reference

| Hook Style | Pattern | Best Platform | Example |
|-----------|---------|--------------|---------|
| Promise | "I [did X]. Here's what I found" | X, LinkedIn | "I analyzed 100 pricing pages. Found 3 patterns" |
| Contrarian | "[Common belief] is wrong. Here's why" | X, LinkedIn | "Everyone tells you to niche down. Here's when you shouldn't" |
| Statistic/Lede | Lead with a surprising number or data point | X, LinkedIn | "97% of cold outreach never gets a reply. The 3% share one trait" |
| Story Hook | Start mid-narrative, create curiosity gap | X, LinkedIn, IG | "We hit $10K MRR. Then almost lost it all" |
| Question | Ask a provocative, audience-relevant question | LinkedIn, IG | "What's the one SaaS metric you track that actually changed your decisions?" |
| Observation | State something obvious that people haven't articulated | X, LinkedIn | "The best founders I know are terrible at celebrating wins" |
| How-to Promise | "How to [achieve specific result] without [common pain]" | LinkedIn, IG, TikTok | "How to price your SaaS without guesswork (the 3-variable model)" |
| Pattern Interrupt | Unusual visual, surprising format, unexpected opener | IG, TikTok | Visual hook: text overlay saying the opposite of what you'd expect |

## Procedure

1. Start with the core idea: what is the ONE thing this post wants to communicate?

2. Identify the target emotion or reaction:
   - Curiosity ("I need to know more")
   - Recognition ("That's exactly my situation")
   - Contrarian intrigue ("Wait, is that true?")
   - Urgency ("I need this now")
   - Validation ("Someone finally said it")

3. Generate hooks across 5-6 styles (selecting the most suitable for the platform and topic):
   - At least 1 Promise-style
   - At least 1 Contrarian-style (even if mild — find an angle)
   - At least 1 Story-style (if applicable)
   - At least 1 Question-style (especially for LinkedIn/IG)
   - At least 1 Statistic/Number-led hook (if data is available)

4. For each hook variation, apply platform formatting:
   - X: First 3-5 words are everything. Max 280 chars total for hook tweet.
   - LinkedIn: First 2 lines (before "see more"). Tease without clickbait.
   - IG: Hook works with visual. First line of caption is critical.
   - TikTok: Text overlay + first 1.5 seconds of audio/visual.

5. Score each hook on:
   - Curiosity gap: does it make you need to know more? (1-10)
   - Specificity: is it concrete vs. generic? (1-10)
   - Brand-fit: does it sound like the brand? (1-10)
   - Platform-fit: does it follow platform rules? (1-10)

6. Rank and recommend top 2 hooks for A/B testing.

## Output Format

```markdown
# Hook Variations: [Post Topic / Platform]

## Core Idea
[One sentence — what this post is about]

## Hook Variations

### 1. [Style: Promise/Contrarian/etc.]
[Hook text, formatted for platform]
**Curiosity:** [1-10] **Specificity:** [1-10] **Brand-fit:** [1-10] **Platform-fit:** [1-10]

### 2. [Style]
...

## Top Picks
1. **Recommended:** [Hook + why — best balance of curiosity and specificity]
2. **A/B challenger:** [Alternate hook + why — different angle worth testing]

## Platform Adaptations
If this hook needs to work across platforms:
- X version: [shorter, punchier]
- LinkedIn version: [slightly more professional framing]
- IG version: [visual-led version]
- TikTok version: [text overlay + hook timing]
```

## Pitfalls

- Hooks that overpromise ("This one trick will 10x your revenue") — kills trust
- Hooks that are all sizzle, no steak (the post doesn't deliver on the promise)
- Question hooks where the audience already knows the answer
- "Clickbait that would embarrass the founder" — a real category you should check against
- Using the same hook style for every post (pattern fatigue)

## Verification

For each top hook: what does this hook promise? Does the actual post deliver that exact promise? If the answer is "sort of" — rewrite either the hook or the post until it's a clean match.
