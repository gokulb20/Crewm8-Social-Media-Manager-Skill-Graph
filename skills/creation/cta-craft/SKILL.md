# CTA Craft

Choose and write the right call-to-action for every post. The CTA is where attention becomes action. A great post with a weak CTA leaves value on the table. A strong CTA on the wrong content type feels desperate. This skill matches CTA to context.

## When to Use

- Finishing any post, thread, or caption
- Testing whether a different CTA would improve conversion
- Content calendar planning (each slot should have a CTA intent)
- Founder says "we're getting engagement but no conversions"

## Quick Reference

| CTA Type | What It Asks For | When to Use | Best Platforms |
|----------|-----------------|-------------|---------------|
| Follow | "Follow @handle for more on [topic]" | End of educational threads, high-value posts | X, TikTok |
| Reply | "What's your experience? Reply below" | Contrarian/opinion posts, questions | X, LinkedIn |
| Save | "Save this for your next [context]" | Frameworks, checklists, reference posts | IG, LinkedIn |
| Share | "Share this with a founder who needs to hear it" | High-value evergreen content | LinkedIn, IG |
| DM | "DM me [word] and I'll send you [resource]" | Lead gen, list building | X, IG |
| Link | "Link in bio" or "Newsletter link in next tweet" | Product, newsletter, website | All (but platform rules vary) |
| Signup | "Join [N] founders getting this weekly" | Newsletter growth, waitlist | LinkedIn, X |
| Poll | "Which do you prefer: A or B?" | Engagement boost, audience research | X, LinkedIn, IG Stories |

## Procedure

1. Determine the post's primary goal:
   - Brand building / authority (no hard ask needed)
   - Engagement boost (reply, poll, share)
   - List growth (signup, DM, link)
   - Product conversion (link, demo signup)
   - Community building (follow, reply, DM)

2. Match CTA intensity to content value delivered:
   - High-value educational post: can ask for save + follow + share
   - Quick observation: "follow" is enough, don't push a signup
   - Hot take: ask for replies (engagement) but don't push a product
   - Product launch: direct CTA is expected and appropriate
   - Rule of thumb: the more value the post delivered, the bigger the ask can be

3. Write the CTA in platform-native format:
   - X: 1 line, no link (unless final tweet of thread). "Follow for more," "Reply with your take"
   - LinkedIn: 1-2 lines. "Agree? Disagree? Let me know below," "Save this for later"
   - IG: 1 line. "Save for later," "Share to your story," "Comment "[X]" below"
   - TikTok: spoken + text overlay. "Follow for part 2," "Link in bio"

4. Test CTA positioning:
   - End of post (standard, expected)
   - Mid-post (bold: "Stop here and save this" — works when the insight is strong enough)
   - No CTA (let content stand alone, works once brand authority is established)

5. Define CTA rotation to avoid fatigue:
   - If every post says "follow for more," followers stop seeing it
   - Rotate through: reply one day, save another, DM another, none another
   - Only hard-ask (signup/link) 1-2x per week maximum

## Output Format

```markdown
# CTA: [Post/Thread Title]

## Post Goal
[Brand building / Engagement / List growth / Product conversion]

## Content Value Delivered
[High / Medium / Low — based on what the post actually teaches or reveals]

## Recommended CTA
[Type + full text, formatted for platform]

## Why This CTA
[Brief note — matches post goal + doesn't over-ask relative to value delivered]

## Alternate CTA (for A/B test)
[Different CTA type + full text]

## CTA Placement
[End of post / Mid-post / No CTA]

## Link (if applicable)
[URL + where it should live per platform rules]
```

## Pitfalls

- Asking for a signup on a post that gave zero value (earn the ask)
- Same CTA every single post (pattern blindness — followers stop seeing it)
- Overcomplicating: 2 CTAs in one post splits attention and reduces action
- No CTA ever (you're building a brand, not a museum)
- CTA that contradicts the post tone (serious post, "LOL DM me" — jarring)

## Verification

Read the post without the CTA. Does it feel complete? Read it with the CTA. Does the ask feel proportionate to the value delivered? If the post is a 2/10 value-wise, the CTA should be a 2/10 ask.
