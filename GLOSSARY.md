# Glossary

Common terms used across Crewm8 Social Skill Graph skills.

| Term | Definition |
|------|-----------|
| **Engagement Rate (ER%)** | (Likes + Comments + Shares + Saves) / Impressions × 100. Measures content quality. |
| **CTA** | Call to Action — the desired action a reader should take after a post (follow, reply, save, click, signup). |
| **Content Pillar** | A repeatable content category that maps to brand positioning (Education, POV, Proof, BTS, CTA). |
| **Hook** | The first sentence of a post that determines whether someone stops scrolling. |
| **Thread** | A series of connected tweets on X, each replying to the previous one. 6-10 tweets optimal. |
| **Auto-Plug** | Adding a link to the first tweet of a thread after it reaches a certain engagement threshold (e.g., 20 likes or 60 minutes). |
| **Carousel** | A multi-slide post format (LinkedIn, IG) where users swipe through content. |
| **Safe Zone** | The area of an image/video guaranteed to be visible on a platform without UI element overlap. |
| **Pillar Mix** | The percentage distribution of content across the brand's pillars (e.g., 35% Education, 25% POV). |
| **Cadence** | Posting frequency per platform (e.g., 2x/day on X, 4x/week on LinkedIn). |
| **Follower Growth Rate** | (Net new followers) / (Total followers) × 100 per period. |
| **Impressions** | How many times content was displayed (not unique views). |
| **Reach** | How many unique accounts saw the content. |
| **Link-in-Bio** | The single clickable link in a social profile bio (only option on IG/TikTok). |
| **Cringe Test** | A pre-publication check: "Would the founder feel embarrassed if this was their top post?" |
| **Brand Voice Flex** | How the brand's voice adapts per-platform (e.g., punchier on X, more structured on LinkedIn). |
| **Voice Drift** | Inconsistency in brand tone across posts or platforms caused by multiple writers or lack of a voice guide. |
| **ER% Baseline** | The average engagement rate over a trailing period (usually 30 days). Used for anomaly detection. |
